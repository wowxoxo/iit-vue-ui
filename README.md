# @iit/vue-ui
It's just a library with some custom Vue components.

## Project setup
```
npm install @iit/vue-ui
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
